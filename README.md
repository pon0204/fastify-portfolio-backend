## アプリ概要
- 自分の考えを深堀りするサービス
- チャットルームを作成し、タイトル,目的,架空の相手を設定する
- 考えを書き出し、送信 → 架空の相手になりきって反論を考え、送信 → 以下繰り返し
- チャットルームは保存出来、ID検索で復元,続きが書ける。
